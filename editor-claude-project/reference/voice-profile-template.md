# Voice Profile Template (L2)

Fill this in once per writer, save it as `voice-profile-<name>.md`, and add it to
the same Claude project as this folder alongside a draft. Everything in brackets is
a prompt, replace it with the writer's own answer and delete the brackets. An
incomplete profile still works: the editor uses whatever fields are filled and
treats blank fields as unknown rather than a licence to guess.

## Basics

- **Writer or project:** [name]
- **Locale and spelling:** [British English, American English, or other, plus any
  local convention worth noting]
- **Register:** [formal, conversational, technical, warm, dry, and so on, in the
  writer's own words]
- **Typical audience:** [who reads this writer, and what they already know]

## Vocabulary

- **Words and phrases this writer actually uses:** [list a handful, with a real
  example sentence for each if you can]
- **Words and phrases this writer never uses:** [the banned list. Be specific,
  "corporate jargon" is too vague, name the actual words]
- **Intensifiers and hedges:** [how this writer adds emphasis or caution, for
  example a word repeated for emphasis rather than swapped for a bigger one, or "I
  think" used as genuine reflection rather than a qualifier]

## Sentence habits

- **Length and rhythm:** [short and clipped, long and winding, deliberately
  uneven, describe the actual pattern rather than an ideal one]
- **Openers:** [how pieces typically start: a memory, a question, a claim, a
  number]
- **Closers:** [how pieces typically end: resolved, open, a callback to the
  opening, a shrug]
- **Punctuation quirks:** [anything specific: fragments used on purpose, no
  semicolons ever, parentheses for asides]

## Structural habits

- **Lists:** [does this writer use bullet points in finished prose, or write in
  paragraphs and reserve lists for reference material]
- **Specificity:** [does the writer name real places, people, dates, and numbers,
  or stay general? Give one real example either way]
- **Self-reference:** [how this writer talks about their own wins and mistakes:
  understated, direct, or largely absent]

## Two short samples

- **A sentence that sounds like this writer:** [paste a real one if possible]
- **A sentence that does not sound like this writer, even though it might pass as
  good writing on its own:** [paste or invent one]

These two samples do more work than every field above put together. When the
fields and the samples disagree, the editor follows the samples.
